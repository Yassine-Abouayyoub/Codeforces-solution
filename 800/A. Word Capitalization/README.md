# Problem: A. Word Capitalization

# Platform
Codeforces

# Language
C

# Topic
Strings, Implementation

# Rating
800

# Problem Description
Capitalization is writing a word with its first letter as a capital letter.

The task is to capitalize the given word while keeping all other letters unchanged.

# Solution Idea
- Read the word.
- Check if the first character is lowercase.
- If it is lowercase, convert it to uppercase using ASCII difference.
- Print the final word.
