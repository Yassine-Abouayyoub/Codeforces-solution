# Problem: A. Petya and Strings

# Platform: Codeforces

# Language: C

# Topic: Strings, Lexicographical Comparison

# Rating: 800

## Problem Description

Little Petya received two strings of equal length consisting of uppercase and lowercase Latin letters.

The task is to compare the strings lexicographically while ignoring letter case differences.

- Uppercase and lowercase versions of the same letter are considered equal.
- Print:
  - `-1` if the first string is smaller
  - `1` if the first string is greater
  - `0` if both strings are equal

## Example

### Input
```text
abs
Abz
