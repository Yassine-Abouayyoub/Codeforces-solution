# Problem: A. Perfect Root

# Platform
Codeforces

# Language
C

# Topic
Math, Implementation

# Rating
800

# Description
For each test case, print `n` distinct positive integers that are considered perfect roots.

Since every positive integer `x` satisfies the condition:
√(x²) = x

all positive integers are valid perfect roots.

# Approach
The solution simply prints the first `n` positive integers:

1 2 3 ... n

They are:
- Positive
- Distinct
- Valid perfect roots

# Complexity
- Time Complexity: O(n)
- Space Complexity: O(1)
